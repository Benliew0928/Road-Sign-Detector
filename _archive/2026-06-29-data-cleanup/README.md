# Data Cleanup Archive

Created: 2026-06-29

This archive contains historical one-off files from the data collection,
ontology review, Roboflow import, P5 cleaning, and Stage C sprint work.

Most files here were moved out of the active project folders to make normal
development easier to understand. The later AI-generated/fake road-sign image
branch was deleted entirely and is not retained in this archive.

## Folders

- `manifests/`: old review CSVs, real/public candidate CSVs, and large detailed
  source manifests.
- `scripts/`: one-off data audit, collection, report generation, and cleanup
  scripts.
- `audit_outputs/`: old JSON/CSV audit outputs and sprint reports.
